# Grundlagen

Shell: ist ein Programm das eine textorientierte Schnittstelle zum Betriebssystem zur Verfügung stellt.

Prompt:

```sh
~/workspace$
```

Einfacher Befehl:

```sh
$ date
```

Jeder Befehl hat Optionen über die das Verhalten gesteuert werden kann.

```sh
$date -I
```

## Optionen
- short hand = 1 Buchstabe
- long hand = Wort

```sh
$date --iso-8601
```

Quelle zu Informationen über einen Befehl:

- häufig: Option `--help`
- man pages (Manual = Handbuch) `$ man date`, (nicht in Replit aber in gängigen Linux Distributionen)
- Internetsuche mit "man date"
- KI

## Argumente

Daten mit denen ein Befehl arbeitet.

```sh
$ echo Hallo Welt
```

`Hallo Welt` ist hier das Argument

mit Optionen 

```sh
$ echo -n Hallo Welt
```

## Tastenkürzel

- Pfeil nach oben/unten: Historie
- Tabulator: automatische File Vervollständigung
- Strg-A: Anfang der Zeile (Strg-E: Ende der Zeile)
- Strg-C: Abbruch des aktuellen Kommandos

## File Befehle

cd, mkdir, ls, touch, pwd, rm, rmdir
working directory (Arbeitsverzeichnis)

Unterschied von absoluter und relativer Pfad

Bedeutung von cd (ohne Argument) und von ~


cd (change directory)

```sh
cd Berichte
```
Ändert es das Vezeichnis auf den Ordner **Berichte**.


mkdir (make directory)

```sh
mkdir abc
```
Erstellt einen Ordner mit dem Namen abc.


ls (list)

```sh
ls 
```
Listet alle Dateien und Ordner des aktuellen Verzeichnis.


touch

```sh
touch xyz
```
Erstellt eine leere Datei mit dem Namen xyz.


pwd (print working directory)

```sh
pwd 
```
Gibt den vollständigen Namen des aktuellen Verzeichnis an.


rm (remove)

```sh
rm xyz
```
Löscht Dateien und Verzeichnisse und ind dem Fall xyz.


rmdir (remove directory)

```sh
rmdir xyz
```
Löscht leere Verzeichnisse und in dem Fall xyz.



Absoluter Pfad:
Zeigt den vollständigen Speicherort einer Datei ab dem **Wurzelverzeichnis**.


Relativer Pfad:
Zeigt den **aktuellen Standort** einer Datei und beschreibt ihn von da aus.


Bedeutung von cd:
cd (change directory) wechselt wieder in das **Home-Verzeichnis** sofern eine anglegt ist.


Bedeutung von ~:
Ist die Bedeutung des Home-Verzeichnisses des aktuellen Benutzers.
