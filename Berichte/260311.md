# Übung 4 
Erkläre folgende begriffe

```sh
|
```
Pipes (|) verbinden Befehle, indem sie die Standardausgabe des linken Befehls als Eingabe an den rechten übergeben.

---

```sh
head
```
head zeigt die ersten 10 Zeilen einer Datei.

---

```sh
grep
```
Mit grep kann man nach bestimmten Zeichen in einer Datei suchen.

---

```sh
tr
```
Mit tr kann man für bestimmte Zeichen was anderes einsetzen.

---

```sh
shuf
```
Mit shuf kannst du die Zeilen einer Datei komplett durchmischen.

```sh
seq
```
seq zählt von einer Zahl bis zu einer anderen. Wenn seq 5 10 dann von 5 bis 10.

---

```sh
date
```
date zeigt dir die aktuelle Systemzeit an.

---

```sh
ROT13
```
ROT13 verschiebt jeden Buchstaben um 13 Stellen nach hinten.

---

```sh
tail
```
tail zeigt dir die letzte Zeile einer Datei.
