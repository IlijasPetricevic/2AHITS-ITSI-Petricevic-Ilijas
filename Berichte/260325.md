## Übung (jne)
Statt auf nop patche den je (jump if equal) Befehl auf einen jne (jump not equal) Befehl. Hintergrund: der jne Befehl macht genau das Gegenteil des je, damit entsteht das gleiche Ergebnis wie durch das nop-patching.
Teste zuerst im GNU Debugger und fixiere den Patch dann im Programm-Binary.
jne = 0x85

```sh
gdb ./pwcheck
b main
run
disas main
x/6xb 0x000055e3373751c8 
set {unsigned char}0x000055e3373751c9 = 0x85
c
```
